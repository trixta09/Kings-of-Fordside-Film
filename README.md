# Harbor Light Film Fundraiser Site

This is a static, dependency-free fundraising website for an independent feature film campaign.

## Files

- `index.html` - page content, sections, donation link, and placeholder team details
- `styles.css` - layout, responsive styling, campaign visuals
- `script.js` - donation amount selector and demo email link behavior
- `assets/harbor-light-key-art.png` - local hero/key-art image

## Quick Customization

1. Replace `Harbor Light` and `HARBOR LIGHT` with the film title.
2. Update the logline, director statement, team names, campaign numbers, and budget rows in `index.html`.
3. Replace `producer@example.com` with the production contact.
4. Change the `#donate-link` URL in `script.js` from the demo `mailto:` link to the live crowdfunding or payment link.
5. Replace `assets/harbor-light-key-art.png` with final poster art or a production still using the same filename.

Open `index.html` directly in a browser to view the site.
